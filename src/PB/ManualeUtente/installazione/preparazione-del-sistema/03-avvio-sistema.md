# Avviare il Sistema

Questa pagina descrive i comandi e le procedure per avviare il sistema in ambiente di sviluppo.

- avviare il broker di messaggi NATS
- avviare il database PostgreSQL / TimescaleDB
- avviare i microservizi: Gateway, Data Consumer, Auth, User, Tenant, Dashboard
- aprire la dashboard web nel browser
- verificare i log e lo stato dei servizi
