# Personalizzare la Configurazione

Questa pagina descrive come configurare le variabili d'ambiente e i file necessari per i microservizi.

- impostare le connessioni al database PostgreSQL/TimescaleDB
- configurare gli endpoint del broker di messaggi NATS
- definire le credenziali per l'autenticazione e i certificati JWT
- configurare i tenant iniziali e i relativi ruoli
- verificare le variabili di ambiente per dashboard, auth, user e gateway
