# Requisiti Software

Requisiti software per lo sviluppo e l'esecuzione del sistema di acquisizione dati.

- sistema operativo Linux o Windows 10+
- Go 1.XX per i microservizi backend
- Node.js / npm per la dashboard frontend
- PostgreSQL e TimescaleDB per la memorizzazione dei dati
- NATS come message broker per la comunicazione tra servizi
- Docker e Docker Compose per ambienti di sviluppo e test
- Grafana e Prometheus per visualizzazione e telemetria
