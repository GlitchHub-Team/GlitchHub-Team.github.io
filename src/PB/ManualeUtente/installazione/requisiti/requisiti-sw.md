# Requisiti Software
Questa sezione descrive i requisiti software necessari per eseguire il Sistema sviluppato da _GlitchHubTeam_ per il [Capitolato C7: Sistema di acquisizione dati da sensori](https://www.math.unipd.it/~tullio/IS-1/2025/Progetto/C7.pdf).

Il software è stato progettato per essere eseguito su qualsiasi sistema operativo che supporti Docker{{gloss}} e Docker Compose{{gloss}}, come ad esempio **Linux**, **Windows** e **MacOS**.

Per avviare il Sistema, perciò è solamente richiesto di installare `Docker Engine` con versione minima `29.0.0`.

Il resto delle dipendenze software sono descritte nel documento di [Specifica Tecnica](https://glitchhub-team.github.io/pdf/PB/DocumentiEsterni/SpecificaTecnica.pdf) per cui si consiglia la lettura.
