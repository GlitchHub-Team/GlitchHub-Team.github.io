# Requisiti Hardware

Questo documento descrive i requisiti hardware minimi e consigliati per eseguire il sistema di acquisizione dati.

- CPU multicore moderna
- almeno 8 GB di RAM per uno sviluppo locale, 16 GB consigliati per test completi
- disco SSD per le prestazioni del database e della cache
- rete locale stabile per la comunicazione tra servizi distribuiti
- spazio libero sufficiente per i dati storici di sensori
