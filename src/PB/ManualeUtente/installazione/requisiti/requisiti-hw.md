# Requisiti Hardware

Questa sezione descrive le caratteristiche consigliate per eseguire il Sistema sviluppato da _GlitchHubTeam_ per il [Capitolato C7: Sistema di acquisizione dati da sensori](https://www.math.unipd.it/~tullio/IS-1/2025/Progetto/C7.pdf).

Il software è stato progettato ed eseguito correttamente all'interno di macchine con le seguenti caratteristiche:
- **Processore**: *Intel Core 7 150U*, *Apple M1*;
- **RAM**: 8 GB;
- **Spazio su disco**: 20 GB di spazio libero.

Le suddette caratteristiche sono strettamente legate alla necessità di eseguire più container Docker{{gloss}} contemporaneamente, i quali richiedono una quantità di risorse non indifferente, soprattutto in termini di spazio su disco.
