# Importare e modificare una dashboard

Questa pagina descrive il processo per importare e personalizzare una dashboard Grafana per il nostro sistema.

- importare il file JSON della dashboard di sistema
- verificare le variabili, le sorgenti dati e gli endpoint
- modificare i pannelli per mostrare le metriche dei servizi e dei gateway
- salvare una copia personalizzata per l'ambiente di test o produzione
