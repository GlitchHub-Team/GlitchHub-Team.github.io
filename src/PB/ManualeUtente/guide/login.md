# Effettuare il login

Questa guida descrive il processo di login per gli utenti del sistema.

## Passaggi principali

1. aprire la pagina di login sulla Dashboard
2. inserire username e password
3. selezionare il tenant corretto, se richiesto
4. completare l'autenticazione

## Comportamento previsto

- il servizio Auth verifica le credenziali
- viene emesso un token JWT
- l'utente viene reindirizzato alla dashboard con l'accesso al tenant appropriato
