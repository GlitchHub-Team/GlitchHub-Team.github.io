# Configurazione Sprint PdP Generator
1. **Set GitHub Token**(opzionale se già fatto): `export PDP_GITHUB_TOKEN= <token presente su BitWarden>`
2. **Navigare fino alla directory dello script**: `cd src/RTB/DocumentiEsterni/sprintPdPGenerator`
3. **Eseguire lo script**: `go run . -sprint X`, dove X è il numero dello sprint
